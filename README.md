# Daily Fashion

---

## Installing

```
# install Vue cli
npm install -g @vue/cli

# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev

# prepare for production via webpack
npm run production

# generate static pages via nuxt.js for static hosting
nuxt generate

```
